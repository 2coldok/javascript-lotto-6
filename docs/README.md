# 기능 구현 목록

- 입력 받는 기능 

- 로또 발행기 (구매 금액을 통해 로또발행 및 출력)

- 로또 ranker : 해당 로또 번호가 몆등인지 계산함

- 로또 중앙 통제기 : 가운데에서 주변 클래스 교통 정리

- 로또 당첨 내역 정리기 : 당첨내역을 잘 포장해서 객체로 반환

## 예외 처리

### 구매 금액 입력(trim()) - 
1. % 1000 = 0 이 아니면 에러
2. 숫자가 아니면 에러

### 당첨 번호 입력(trim()) - 
1. 1~45 의 숫자가 이외의 요소이면 에러
2. 길이 6이 아니면 에러

### 보너스 번호 입력(trim()) - 
1. 당첨 번호와 중복시 에러
2. 1~45 이외의 숫자시 에러
